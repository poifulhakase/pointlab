## 1. プロダクト定義

株式投資家向けの Web カレンダー / 需給分析 PWA「ぽいロボ」。配当日・SQ日・マクロイベント・市場休場日を表示し、メモ・チャート・需給分析・AI連携プロンプトを提供する。

- **本番URL**: https://pointlab.vercel.app/stock-calendar
- **対象ユーザー**: 当面は友人限定クローズドβ（無料）。

### 法務スタンス（要件レベルで遵守）
- 投資助言業の**登録はしない**。客観データ・教育・ツール・コミュニティの提供に徹する。
- エンジン/シールドの出力は**客観的状態記述型**（命令・個別銘柄推奨・売買指示をしない）。最終判断はユーザー。
- 詳細: `docs/legal/`（業務範囲・金商法自己点検・利用規約・特商法・免責・課金導線）。

## 3. 主要機能要件

### カレンダー
- 月/週/日ビュー切替。配当・SQ・FOMC/雇用統計(NFP)/CPI/PCE/ISM/日銀/短観等のマクロイベント、市場休場日、アノマリーを表示。
- 自由メモ（スティッキー）、日別スケジュール帯（1日複数可）。
- **ぽいロボレーダー**: 選択したイベント種別の日付セルをハイライト（デフォルトOFF・localStorage永続化）。

### エンジン（需給分析）
- 4タブ: 分析 / 環境 / 現物 / 先物。VIX・NT倍率（日経÷TOPIX）・信用倍率・投資主体別・先物手口・PCR 等を表示。
- **TEV（需給エネルギー）**: `src/utils/tevCore.mjs` が計算式の単一情報源（logic_version `v2-symmetric-restoring`）。
- **AI連携プロンプト出力** (`engineExport.ts` / `buildExportJson`): 需給データ＋プロンプトをコピーし Claude/Gemini/ChatGPT/DeepSeek へ。出力は状態記述型。
- **バックテスト** (`backtest-tev.mjs` / BacktestPanel): TEVシグナルの過去勝率を検証。確信度キャリブレーション付き。

### シールド
- イベント予想プロンプト（`newsPrompt.ts`）。イベント=需給×価格レジームの点火装置として影響寿命・トリガー種別を付記。

### 研究室（ランディング/設定/資料）
- 設定: テーマ・通知(FCM)・表示プレビュー・メンテナンスモード（管理者）。
- 資料: 仕様書(SpecView)・利用規約・免責・バックテスト・プロンプトEvals(管理者)・戦略プレイブック。
- **ぽいロボコネクト**（予約・通話）: JaaS(Jitsi) による通話。**会員限定**。
- コミュニティメンバー管理（管理者）。

## 4. 非機能要件・制約

- **PWA**: インストール可能。データ鮮度のため `public/calendar/data/` は NetworkFirst。precache は骨格のみ（遅延チャンクはランタイムキャッシュ）。
- **データ更新**: `.github/workflows/fetch-data.yml` に一本化。平日 19:30/21:30 JST 取得、土曜 09:00 週次・バックテスト・更新通知。main push で Vercel 自動デプロイ。
- **認証/権限**: Google ログイン。管理者は単一メール（`firestore.rules` の `isAdmin()` が正）。ログインなしでもアプリには入れる（会員制は各ビューのロック画面で表現）。
- **データ永続化**: localStorage ＋ Firestore（ログイン時クロスデバイス同期）。書き込みは REST 経由（SDK 書き込みはハングするため不採用）。
- **エラー監視**: Sentry（遅延 init・ソースマップ対応）。
- **セキュリティ**: API のエラーメッセージは固定文言（`e.message` を漏らさない）。Firestore ルールでアクセス制御。

## 5. 投資ロジックの設計思想（背景）

- 目標は「低ドローダウン × スイング」。トレンドフィルター必須（逆張りは確定下落局面でのみ）。
- 20年/52週R&Dの結論: CAGR と DD は両立に限界があり、現実解は CAGR 15〜20% × DD −40% 圏（2倍レバ前提）。
- ブル/ベア 1倍2倍 ETF（現物）でシグナルを表現。レバレッジ運用は個人口座戦略として別管理。
- **確信度は「当たる保証」ではない**。小標本（n≈27〜52週）のためシュリンク重視。トレンド逆行シグナルは確信度を抑制する（数値ゲート＋プロンプト両面）。
