## 1. 共通仕様

### 1.1 ビューポート・レイアウト

| 項目 | 仕様 |
|------|------|
| 対象デバイス | スマートフォン（モバイルファースト） |
| viewport | `width=device-width, initial-scale=1.0`（一部ページは `maximum-scale=1.0, user-scalable=no`） |
| 最大幅 | 500px（中央寄せ、`margin: 0 auto`） |
| 地図ページ | PC時（min-width: 501px）は横幅全画面、最大幅制限なし |

### 1.2 セキュリティ・メタ

- `X-Content-Type-Options: nosniff`
- `referrer: strict-origin-when-cross-origin`
- コンテキストメニュー（長押し）無効化（全ページ）
- base href: `/poinavi/`

### 1.3 アクセシビリティ

- 主要ラベル・ボタンに `aria-label` を付与
- 装飾要素に `aria-hidden="true"`
- 非表示テキストに `.visually-hidden` クラス使用

## 2. ラボノート（index.html）

**用途**: 旅先のメモ・ノートを保存。オフライン対応。

### 2.1 レイアウト構成

```
┌─────────────────────────────────┐
│ [ヘッダー] ぽいナビロゴ  [❓][⚙] │
├─────────────────────────────────┤
│                                 │
│  [メモ入力エリア]               │
│  ┌───────────────────────────┐  │
│  │ メモを入力...             │  │
│  └───────────────────────────┘  │
│  [追加] ボタン                   │
│                                 │
│  [メモ一覧]                     │
│  ┌───────────────────────────┐  │
│  │ 3/15 12:34    [地図][📋][🗑]│  │
│  │ メモ本文...                │  │
│  │ [添付画像サムネイル]       │  │
│  └───────────────────────────┘  │
│  ...                            │
│                                 │
├─────────────────────────────────┤
│ [探索室][ラボノート][翻訳][ぽいナビ]│
└─────────────────────────────────┘
```

### 2.2 機能一覧

| 機能 | 説明 | 備考 |
|------|------|------|
| メモ追加 | テキスト入力＋画像添付（最大複数枚） | ローカルStorage保存 |
| メモ一覧 | 作成日時降順、コピー・削除ボタン | 画像はlazy loading |
| 地図リンク | 位置情報付きメモのみ「地図」ボタン表示 | map.html に座標パラメータで遷移 |
| URL自動リンク化 | メモ内のURLをクリック可能なリンクに変換 | XSS対策でescapeHtml適用 |

### 2.3 設定モーダル項目

- **トップ表示**: 起動時に表示するページ（ラボノート/翻訳マシン/ぽいナビ/研究室）
- **テーマ**: ライトモード / ダークモード
- **データ管理**: 設定を初期化
- **免責事項**: テキスト表示

## 3. 研究室（lab.html）

**用途**: 天気・歩数・移動距離のダッシュボード。旅先の体調管理に。

### 3.1 レイアウト構成

```
┌─────────────────────────────────┐
│ [ヘッダー] ぽいナビロゴ  [❓][⚙] │
├─────────────────────────────────┤
│                                 │
│  [日付・時刻]                   │
│      11:34                      │
│    3/15 にちようび               │
│                                 │
│  [天気カード] [気温カード]      │
│  （2列グリッド）                │
│                                 │
│  [歩数・移動距離パネル]         │
│  ┌─────────┬─────────┐         │
│  │ 歩数    │ 移動距離 │         │
│  │ xxx歩   │ x.x km  │         │
│  └─────────┴─────────┘         │
│                                 │
│  [キャラクター表示エリア]       │
│  博士イラスト + ぽいろぼ         │
│  （ダークモード時は白浮遊物）   │
│                                 │
├─────────────────────────────────┤
│ [探索室][ラボノート][翻訳][ぽいナビ]│
└─────────────────────────────────┘
```

### 3.2 機能一覧

| 機能 | 説明 | データソース |
|------|------|--------------|
| 日付・時刻 | リアルタイム表示、曜日ひらがな表記 | ローカル |
| 天気 | 現在の天気アイコン＋説明文 | Open-Meteo API |
| 気温 | 現在の気温 | Open-Meteo API |
| 歩数 | 本日の歩数 | Fitbit API（要連携） |
| 移動距離 | 本日の移動距離 | GPS位置情報 |
| 波紋エフェクト | メインエリアタップ時の視覚的フィードバック | - |
| 白い浮遊パーティクル | ダークモード時のみ、背景に25個の粒子 | opacityアニメーション、枠内に収納 |

### 3.3 背景・フォント

- **ライトモード**: `lab-bg-light.png` ＋ 白グラデーション
- **ダークモード**: `lab-bg.png` （研究所イラスト）
- **フォント**: Cherry Bomb One（Google Fonts、display=swap）※日付・時刻エリア
- パーティクル: body直下に配置、`position: fixed`、`clip-path: inset(0)` で枠内に収納

### 3.5 現在の表示状態（ lab.html 内スタイルによる）

- **天気・気温・歩数・移動距離**: 一時非表示（`display: none !important`）の設定あり。デザイン改修後に再表示予定。
- 日付・時刻、キャラクター表示、フッターは通常表示。

### 3.6 設定モーダル項目

- **トップ表示**: index / translate / map / lab から選択
- **テーマ**: ライト / ダーク
- **初心者ガイド**: ON/OFF トグル（各画面の使い方アイコン表示制御）
- **データ管理**: 設定を初期化
- **免責事項** / **データソース**: テキスト表示

## 4. 翻訳マシン（translate.html）

**用途**: テキスト翻訳、カメラOCR、音声入力、通貨・単位変換。

### 4.1 レイアウト構成

```
┌─────────────────────────────────┐
│ [ヘッダー] ぽいナビロゴ  [❓][⚙] │
├─────────────────────────────────┤
│                                 │
│  [通貨][音声翻訳][単位]  ←中央   │
│  （大きな円形ボタン）            │
│                                 │
│  [言語選択] 自分: ○○ / 相手: ○○│
│                                 │
│  [翻訳結果一覧]                 │
│  ┌───────────────────────────┐  │
│  │ 原文 / 訳文 / ソース種別  │  │
│  └───────────────────────────┘  │
│                                 │
├─────────────────────────────────┤
│ [探索室][ラボノート][翻訳][ぽいナビ]│
└─────────────────────────────────┘
```

### 4.2 機能一覧

| 機能 | 説明 | データソース |
|------|------|--------------|
| テキスト翻訳 | 入力→翻訳、100言語以上 | LibreTranslate, MyMemory |
| 会話モード | 自分・相手の言語設定、音声入出力 | Web Speech API |
| カメラOCR | カメラ撮影→OCR→翻訳 | Tesseract.js（遅延読み込み） |
| 画像翻訳 | 画像アップロード→OCR→翻訳 | Tesseract.js |
| 通貨変換 | 金額＋通貨ペアで変換 | Frankfurter API |
| 単位変換 | 長さ・重さ等の単位変換 | ローカル計算 |
| QRコード読み取り | カメラでQR読み取り | jsQR |

### 4.3 設定モーダル項目

- **会話モード**: 自分・相手の言語
- **カメラ・画像翻訳**: OCR言語
- **通貨**: 基準通貨
- **単位**: 長さ・重さ等の単位系
- **初心者ガイド**: ON/OFF
- **トップ表示** / **テーマ** / **免責事項** / **データソース**

### 4.4 パフォーマンス

- Tesseract.js: 初回OCR使用時にのみ動的読み込み（約2MB）

## 5. ぽいナビ地図（map.html）

**用途**: 現在地周辺の店舗・駅・公共トイレ検索。Google Maps 連携。

### 5.1 レイアウト構成

```
┌─────────────────────────────────┐
│                     [❓] [⚙]   │  ← 右上フロート
│  [レイヤーコントロール・左上]   │
│  [トイレ] [路線] [雨雲]         │
│                                 │
│  [マップ表示エリア]             │
│  ┌───────────────────────────┐  │
│  │                           │  │
│  │     Google Map            │  │
│  │                           │  │
│  └───────────────────────────┘  │
│                                 │
│  [検索入力エリア・下部]         │
│  ┌───────────────────────────┐  │
│  │ 研究所 [検索]              │  │
│  │ [タグ選択エリア]           │  │
│  └───────────────────────────┘  │
│                                 │
├─────────────────────────────────┤
│ [探索室][ラボノート][翻訳][ぽいナビ]│
└─────────────────────────────────┘
```

### 5.2 機能一覧

| 機能 | 説明 | データソース |
|------|------|--------------|
| マップ表示 | 現在地中心、ズーム・ドラッグ | Google Maps API |
| 周辺検索 | テキスト＋タグで検索 | Google Places API |
| タグ管理 | 複数タグの追加・削除・選択 | ローカルStorage |
| お手洗いレイヤー | 半径2kmの公共トイレ表示 | 国土交通省API |
| 鉄道路線レイヤー | 半径8kmの路線表示 | Overpass API |
| 雨雲レーダー | 降雨レイヤー表示 | RainViewer API |
| 検索結果リスト | トップ件数を距離順表示 | - |

### 5.3 ヘッダー・UIの違い

- **ヘッダー**: translate-header ではなく、マップ上に `map-header-icons` で初心者・設定ボタンをフロート表示
- **アイコン間隔**: ガイド（電球）と設定（歯車）の gap は `var(--s16)`（翻訳マシン `.translate-header__icons` に合わせる）
- **設定モーダル**: `settings-modal` クラス（他は `translate-modal`）
- **設定項目**: タグ管理、トップ表示、テーマ、初心者ガイド、営業中のみ表示、交通手段、免責事項、データソース

### 5.4 フッター

- 4タブナビ（研究室・ラボノート・翻訳マシン・ぽいナビ）
- 現在ページは `translate-footer__btn--active` でハイライト

## 6. 共通コンポーネント

### 6.1 ヘッダー（translate-header）

**対象**: index, lab, translate（mapは別レイアウト）

| 要素 | 仕様 |
|------|------|
| 左 | スペーサー（visibility: hidden、レイアウト用） |
| 中央 | **ぽいナビロゴ**（poinabi.svg）・常に画面横幅の中央に配置（`position: absolute; left: 50%; transform: translateX(-50%)`） |
| 右 | 初心者アイコン（❓）＋ 設定アイコン（⚙） |
| ロゴリンク | note.com/pointlab/magazines |
| 初心者アイコン | 設定で非表示可能（visibility: hidden でスペースは維持） |

### 6.2 フッター（translate-footer）

**対象**: 全4ページ

| 項目 | 仕様 |
|------|------|
| 高さ | `68px + env(safe-area-inset-bottom)` |
| 配置 | 4タブ均等、`justify-content: space-around` |
| タブ | 探索室(lab) / ラボノート(index) / 翻訳マシン(translate) / ぽいナビ(map) |
| 研究室ページ | `position: fixed; bottom: 0` で常時表示 |
| スマホ（≤768px） | 左右余白なし（`max-width: 100%`、研究室は `left: 0; right: 0` で全幅） |
| PC（≥769px） | 最大幅 500px 中央寄せ |
| スタイル | ライト時に `background: rgba(255,255,255,0.75); backdrop-filter: blur(8px)` |

### 6.3 設定モーダル（translate-modal / settings-modal）

| 項目 | 仕様 |
|------|------|
| オーバーレイ | クリックで閉じる |
| 閉じるボタン | 右上 × アイコン |
| コンテンツ | スクロール可能、最大高さ制限 |
| 一貫性 | 各ページでIDが異なる（memoSettingsModal, labSettingsModal, translateSettingsModal, settingsModal） |

### 6.4 アラート・確認モーダル（poinavi-alert-modal）

- `poinaviAlert(message)` / `poinaviConfirm(message)` で利用
- システム alert/confirm の代替

### 6.5 初心者ガイド（beginner-tour）

- ステップ式オーバーレイ、スポットライトで要素を強調
- 右上 × アイコンで閉じる
- スポットライト位置に応じて説明カードを上下に自動配置（背後を隠さない）
- 設定の「初心者ガイド」トグルでアイコン表示をON/OFF

## 7. テーマ・カラーモード

### 7.1 モード

| モード | html/body クラス | 主な背景色 |
|--------|------------------|-----------|
| ライト | `light-mode` | `#f5f5f7` |
| ダーク | `dark-mode` | `#1a1a1a` |

### 7.2 保存・適用

- **キー**: `localStorage.poinavi_theme`（"light" / "dark"）
- **適用タイミング**: ページ読み込み前（FOUC対策のインラインスクリプト）、ページ遷移時、bfcache復元時
- **共通ユーティリティ**: `PoinaviShared.getTheme()` / `setTheme()` / `applyTheme()`

### 7.3 カスタムプロパティ（抜粋）

- `--text-primary`, `--text-secondary`, `--bg-primary`, `--bg-secondary`
- `--border-color`, `--accent-color`, `--accent-hover`
