## なぜこの作業をしたか（背景）

有料会員（予定）×「ロボのシグナルをスイングの補助に」というモデルは、
**有償の売買シグナル配信＝投資助言・代理業（金融商品取引業）の登録が必要になる可能性が高い**領域。
金融庁・証券取引等監視委員会による無登録営業の摘発実績がある。

→ 方針は **B案**（ロボは客観データ・指標・過去検証の提示と教育に徹し、特定の売買を推奨/配信しない／
判断はユーザー自身／課金はツール・閲覧・教育・コミュニティの対価）。
免責（アプリ内 LegalModal）と戦略プレイブックには既に前さばきの文言を反映済み（commit 6515e64）。

本フォルダは、その B案を**最終確定**し、付随する法務文書を一式そろえて、
専門家レビューに最短で進めるための準備物。
