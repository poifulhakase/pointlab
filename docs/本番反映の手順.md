# 本番反映の手順（pointlab.vercel.app）

## 現状

- **ローカル**（このワークスペース）: 博士のおすすめ・フル名称・ファビコン更新済み
- **本番**: GitHub の内容を Vercel が自動デプロイ
- **問題**: ローカル変更が GitHub に push されていないため、本番が古いまま

## 反映方法

### 方法1: sync-and-push.bat（推奨）

1. **副業画像**を `pointlab/images/Side_Biz_Encyclopedia_Delegate.png` または `pointlab/images-source/Side_Biz_Encyclopedia_Delegate.png` に配置
2. エクスプローラーで `pointlab\batch` フォルダを開く
3. **sync-and-push.bat** を**ダブルクリック**して実行（画像コピー＋キャッシュバスティングを自動実行）
4. 変更内容を確認し、`y` を入力して Enter
5. GitHub に push されると Vercel が自動デプロイ（1〜2分）
6. https://pointlab.vercel.app で確認

※ 必ず**ダブルクリック**で実行してください。ターミナルから実行すると日本語パスで失敗することがあります。
※ 本番とローカルで画像が異なる場合：images-source\Side_Biz_Encyclopedia_Delegate.png を置き、sync-and-push を再実行。
※ favicon/副業画像が変わらない場合：ブラウザで Ctrl+Shift+R（ハードリロード）を試す。

### 方法2: GitHub 上で直接編集

1. https://github.com/poifulhakase/pointlab を開く
2. `index.html` の編集（鉛筆アイコン）
3. 「おすすめ」→「博士のおすすめ」など必要な箇所を編集
4. コミットして push

### 方法3: 手動コピー＆push

1. `pointlab` フォルダの内容を `C:\PointLabSync` にコピー（.git, node_modules 除く）
2. `C:\PointLabSync` で `git add -A && git commit -m "Update" && git push origin main`

---

## deploy-now.bat について

`deploy-now.bat` は Vercel CLI でローカルをデプロイしますが、**Vercel のプロダクションは GitHub 連携が優先**される場合があります。確実に反映するには、まず GitHub に push してください。
