# Vercel「Deploy manually」の見つけ方

## 📍 「Deploy manually」の場所

### 方法1：画面を下にスクロール
現在の画面を**下にスクロール**して「Deploy manually」セクションを探す

### 方法2：URLから直接アクセス（推奨）
```
https://vercel.com/new?mode=manual
```

### 方法3：GitHubリポジトリでインポート
GitHubでリポジトリを作成してVercelでインポートする方法が確実です。

---

## 🌐 代替：Netlify

Netlifyの方が「Deploy manually」が見つかりやすい場合があります：
1. https://www.netlify.com にアクセス
2. 「Add new site」→「Deploy manually」
