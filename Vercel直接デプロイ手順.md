# Vercel直接デプロイ手順

## 🎯 最も簡単な方法

以下のURLに直接アクセスしてください：

```
https://vercel.com/new?mode=manual
```

このURLで「Deploy manually」画面が直接開きます。

---

## 📦 デプロイ手順

1. 上記URLにアクセス
2. ドラッグ&ドロップエリアが表示される
3. 以下のフォルダをドラッグ&ドロップ：
   ```
   C:\Users\owner\OneDrive\デスクトップ\PointLab
   ```
4. 数秒でデプロイが完了
5. URLが発行されます（例: `https://pointlab.vercel.app`）

---

## 🌐 ドメイン設定（後で）

デプロイ後、Vercelのダッシュボードで：
1. プロジェクトを選択
2. 「Settings」→「Domains」
3. `pointlab` を入力して追加

---

## ⚠️ もし上記URLでも見つからない場合

**Netlifyを使用**（以前成功した方法）：
1. https://www.netlify.com にアクセス
2. 「Add new site」→「Deploy manually」
3. フォルダをドラッグ&ドロップ

Netlifyの方が手動デプロイが見つかりやすいです。
