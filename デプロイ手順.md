# デプロイ手順（Vercel / Netlify）

## 📋 前提条件

- GitHubアカウント（推奨：Git連携で自動デプロイ）
- または、Vercel / Netlifyアカウント

## 🚀 方法1：Vercel（推奨）

### ステップ1：Vercelアカウント作成
1. https://vercel.com にアクセス
2. 「Sign Up」→ GitHubアカウントでログイン（推奨）

### ステップ2：プロジェクトをデプロイ

#### オプションA：GitHub経由（自動デプロイ）
1. GitHubでリポジトリを作成
2. 現在のファイルをコミット・プッシュ
3. Vercelで「Import Project」→ GitHubリポジトリを選択
4. 設定：
   - Framework Preset: **Other**
   - Build Command: （空欄のまま）
   - Output Directory: （空欄のまま）
   - Install Command: （空欄のまま）
5. 「Deploy」をクリック

#### オプションB：ドラッグ&ドロップ（手動デプロイ）
1. Vercelダッシュボードで「Add New Project」
2. 「Deploy manually」を選択
3. 現在のフォルダ（PointLab）をドラッグ&ドロップ
4. 数秒でURLが発行されます（例: `https://pointlab-xxxxx.vercel.app`）

### ステップ3：カスタムドメイン設定
1. Vercelダッシュボード → プロジェクト → Settings → Domains
2. 「Add Domain」をクリック
3. 取得済みのドメイン（例: `pointlab.com`）を入力
4. DNS設定の指示に従って設定
   - レコードタイプ: `CNAME` または `A`
   - 値: Vercelが提示する値

## 🌐 方法2：Netlify

### ステップ1：Netlifyアカウント作成
1. https://www.netlify.com にアクセス
2. 「Sign up」→ GitHubアカウントでログイン（推奨）

### ステップ2：プロジェクトをデプロイ

#### オプションA：GitHub経由（自動デプロイ）
1. GitHubでリポジトリを作成
2. 現在のファイルをコミット・プッシュ
3. Netlifyで「Add new site」→ 「Import an existing project」
4. GitHubリポジトリを選択
5. 設定：
   - Build command: （空欄のまま）
   - Publish directory: （空欄のまま）
6. 「Deploy site」をクリック

#### オプションB：ドラッグ&ドロップ（手動デプロイ）
1. Netlifyダッシュボードで「Add new site」→ 「Deploy manually」
2. 現在のフォルダ（PointLab）をドラッグ&ドロップ
3. 数秒でURLが発行されます（例: `https://pointlab-xxxxx.netlify.app`）

### ステップ3：カスタムドメイン設定
1. Netlifyダッシュボード → Site settings → Domain management
2. 「Add custom domain」をクリック
3. 取得済みのドメイン（例: `pointlab.com`）を入力
4. DNS設定の指示に従って設定
   - レコードタイプ: `CNAME` または `A`
   - 値: Netlifyが提示する値

## 🔧 カスタムドメイン取得について

### ドメイン取得サービス
- **お名前.com**: https://www.onamae.com
- **ムームードメイン**: https://muumuu-domain.com
- **Google Domains**: https://domains.google
- **Namecheap**: https://www.namecheap.com

### 推奨ドメイン
- `pointlab.com`（.comが最も一般的）
- `pointlab.jp`（日本向け）
- `pointlab.net`（代替案）

### ドメイン取得後の設定
1. ドメイン取得サービスでDNS設定画面を開く
2. Vercel / Netlifyが提示するDNSレコードを追加
3. 反映まで数時間〜24時間かかる場合があります

## 🎯 推奨：GitHub連携

### メリット
- ファイル変更を自動でデプロイ
- バージョン管理ができる
- 複数人で作業しやすい

### 手順
1. GitHubでリポジトリ作成
2. ローカルで以下を実行：
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/your-username/pointlab.git
   git push -u origin main
   ```
3. Vercel / NetlifyでGitHubリポジトリを連携
