事業内容

「小さく始めて、必要に応じて育てていけるWebサイト」をコンセプトに、
個人事業主の方向けの静的Webサイト制作と、情報発信まわりの整理を行っています。

サービス1：名刺代わりの5ページWebサイト制作

まずは「きちんとした公式サイト」を持ちたい方向けに、5ページ構成のシンプルなWebサイトを制作します。

できること

  • トップ／事業内容／プロフィール／実績・活動／お問い合わせの構成設計
  • HTML / CSS による静的Webサイト制作
  • Vercel / Netlify など静的ホスティングサービスへの公開サポート（手順のご案内）
  • 文章・情報の整理（ヒアリング内容をもとに、たたき台を作成）

できないこと

  • 複雑なシステム開発（会員サイト・予約システムなどの実装）
  • 高度なアニメーション・リッチなUIを前提としたデザイン制作
  • 継続的な運用代行・更新代行（都度のご相談は可能です）

向いている人

  • まずは最低限の情報で「公式サイト」を整えたい個人事業主の方
  • 自分で更新する予定は少なく、しばらくは名刺代わりとして使いたい方
  • 将来リニューアルや拡張を見据えて、土台をシンプルに整えたい方

サービス2：文章と情報構成の整理

すでに発信している情報（ブログ・SNS・資料など）をもとに、Webサイトに載せる内容をコンパクトに整理します。

できること

  • 事業内容・プロフィール・実績などの項目設計
  • 「誰に・何を・どのように」伝えるかの文章のたたき台作成
  • 必要最小限のキーワード整理（将来のSEO対策に向けた土台づくり）

できないこと

  • 専門のライターとしての大規模な取材・インタビュー
  • 検索順位の保証や、短期間での大幅な集客アップ

向いている人

  • 自分で文章を書くのが少し苦手、または時間が取りづらい方
  • 発信内容が増えすぎて、一度整理したいと感じている方

サービス3：今後の発信・サイト運用の簡易アドバイス

制作後の「どのページを増やすとよいか」「どんな発信がサイトと相性がよいか」など、運用の方向性を一緒に整理します。

できること

  • 個人事業主の状況に合わせた、発信の優先順位の整理
  • note / X（旧Twitter）など外部サービスとの連携イメージのご提案
  • 「今はやらないこと」を決めるための簡易な整理

できないこと

  • 日々の投稿代行や、細かな運用レポートの作成
  • 広告運用やマーケティングの専門的なコンサルティング

向いている人

  • 仕事の合間に、無理なく発信を続けたい個人事業主の方
  • 「全部やろう」とするのではなく、少しずつ整えていきたい方

まずは状況のヒアリングから

いきなり制作のご相談ではなく、「今の状況だと、どこから整えるのが良さそうか」を一緒に整理するところからでも大丈夫です。
