# 🎓 ぽいふる博士のお金相談室

節約・ポイント活用について相談できるAIチャットボットです。

## セットアップ

### 1. 依存パッケージのインストール

```bash
cd hakaseAI
npm install
```

### 2. 環境変数の設定

プロジェクトルートに `.env` ファイルを作成：

```
OPENAI_API_KEY=sk-your-api-key-here
PORT=3001
```

※ APIキーは https://platform.openai.com/api-keys から取得

### 3. サーバー起動

```bash
npm start
```

ブラウザで http://localhost:3001 にアクセス

## APIキーなしでも動作します

OpenAI APIキーが設定されていない場合、モックモードで動作します。
基本的な節約相談には応答できます。

## 機能

- 💬 チャット形式でお金の相談
- 🎯 節約・ポイント活用に特化
- 📝 会話履歴を考慮した回答
- 🎨 温かみのあるUI

## 博士の特徴

- 博士口調（〜じゃ、〜かもしれん）
- 断定しない、参考情報として回答
- 1〜3文で簡潔に回答
